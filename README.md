### 反射机制
> class的本质是一种数据类型
> 无继承关系的数据类型无法赋值
> 反射是JVM等级的性质, 所以程序上JVM编译的时候可以忽略一些import依赖关系

JVM在第一次读取到一种class的时候, 就创建一个Class类实例

JVM持有的每个Class实例都指向同一个数据类型
